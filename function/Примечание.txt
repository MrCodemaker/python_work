ПРИМЕЧАНИЕ
1) Если вы используете значения по умолчанию, все параметры со значением по умолчанию должны следовать после параметров, у которых значений по умолчанию нет . Это необходимо для того, чтобы Python правильно интерпретировал позиционные аргументы .

2) Первый способ импортирования, при котором записывается команда import с именем модуля, открывает доступ программе ко всем функциям из модуля: имя_модуля.имя_функции()

3) Также возможно импортировать конкретную функцию из модуля. Общий синтаксис выглядит так: from имя_модуля import имя_функции Вы можете импортировать любое количество функций из модуля, разделив их имена запятыми: from имя_модуля import функция_0, функция_1, функция_2
